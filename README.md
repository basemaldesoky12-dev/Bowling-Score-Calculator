# Bowling Score Calculator

A clean, maintainable **Node.js utility** for calculating bowling scores based on a series of frames.  
This project demonstrates efficient algorithmic logic for handling **strikes, spares, and lookahead bonuses** without using nested loops or recursion.

---

## Features


- **CLI Support**  
  Run calculations directly from the command line with custom inputs.

- **Unit Tests**  
  Comprehensive Jest test suite covering edge cases (perfect games, turkeys, partial games).

- **Clean Architecture**  
  Separated logic (`BowlingGame` class) and entry point.
